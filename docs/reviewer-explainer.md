# Very Simple Reviewer Explanation

## 20-second version

> This demo lets you talk to a photo library the way you remember moments in real life. Instead of searching by filename or date, you can ask for a memory in plain language, like a trip, a person, or a feeling, and the system tries to bring back the most relevant photos.

## 45-second version

> Think of this as a smart photo assistant. You open the link, ask for a memory in natural language, and the system interprets what you mean, searches the demo photo library, and shows the best matches. If your request is too broad, it asks a quick follow-up rather than guessing. The point of the prototype is to test whether conversational photo recall feels natural, fast, and useful in an Alexa+-style experience.

## 90-second version

> The reviewer experience is intentionally simple. You open a public link and land in a ready-to-use photo library. From there, you can type or speak a memory request, like “show our Yosemite trip last June” or “find photos of Jake playing soccer.” Behind the scenes, the system translates that request into useful search signals, looks through the memory library, and returns the closest matches. If there is ambiguity, like multiple beach trips, it pauses and asks which one you meant. That helps the demo feel more conversational and less like a brittle keyword search tool.

## If someone asks “why this matters”

> People do not remember photos as file names or folders. They remember stories, people, emotions, and moments. This demo is trying to prove that a voice-first, conversational interface can make rediscovering memories feel much more natural.

## If someone asks “what is special here”

> The important thing is not just showing photos. It is the loop of understanding a natural request, keeping context across follow-up questions, and recovering gracefully when the request is unclear.
